Para realizar a instalação do Cypress e Playwright é necessário ter o Node.js instalado:

    - 3.1 Node Js instalado
        - 3.1.A - Dê preferência para versão LTS - Long time support
        - 3.1.B - https://nodejs.org/en/download
        - 3.1.C - Atentar-se em permissão para ser adicionado ao path
        - 3.1.D - Para verificar a versão windows + r ou pesquisar por cmd
            Digita cmd
                Digita node --version
                Digita node -v
